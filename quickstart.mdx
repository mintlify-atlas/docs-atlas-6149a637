# Get Started with Aria

Welcome to Aria! This guide will help you understand the platform and join our waitlist to be among the first users of India's most advanced digital healthcare platform.

<Info>
  Aria is currently in development. Join our waitlist to get early access when we launch.
</Info>

## Understanding Aria

Aria is a dual-app system designed for both healthcare providers and patients:

<CardGroup cols={2}>
  <Card title="Doctor App" icon="stethoscope">
    Create prescriptions with AI-powered dictation, access complete patient histories, and manage your practice digitally
  </Card>
  
  <Card title="Patient App" icon="mobile-screen">
    Access all your medical records, prescriptions, and health data in one secure location
  </Card>
</CardGroup>

## Platform Overview

### Dual App System

Dedicated apps for doctors and patients with secure role-based access and seamless identity management.

```tsx
// User type selection from waitlist form
const userTypes = {
  doctor: "Healthcare provider creating prescriptions",
  patient: "Individual managing personal health records",
  other: "Healthcare organization or related service"
};
```

### Digital Prescriptions

Create, view, and manage digital prescriptions with complete history accessible to both doctor and patient anytime.

<Tip>
  Every prescription includes diagnosis, medications with dosages, and treatment duration - all stored securely in the cloud.
</Tip>

### Medical Record Management

Patients can upload lab scans, hospital reports, and documents for a richer medical history and better care.

### AI & Analytics Ready

Built for future AI integrations to automatically order medicines, find patterns, and provide intelligent insights.

## Key Features to Explore

When Aria launches, you'll have access to these powerful features:

### 1. Medical Voice Dictation

Real-time prescription writing through voice with automatic entity recognition.

```tsx
// Example dictation input
"Patient complains of fever and headache for 3 days. 
Prescribing Paracetamol 500mg and Cetirizine 10mg for symptom relief."

// Automatically detected entities:
// - Symptoms: fever, headache
// - Medications: Paracetamol 500mg, Cetirizine 10mg
// - Duration: 3 days
```

<Tip>
  The AI recognizes medical terminology, drug names, dosages, and symptoms automatically - no manual tagging needed.
</Tip>

### 2. Smart Medication Search

Instant medication lookup with ABHA-compliant database:

```tsx
// Search "croc" returns:
[
  { name: "Crocin 500mg", composition: "Paracetamol" },
  { name: "Crocin 650mg", composition: "Paracetamol" },
  { name: "Crocin Advance 500mg", composition: "Paracetamol + Caffeine" }
]
```

### 3. Complete Patient Timeline

Chronological view of all medical records:

- Prescriptions with full diagnosis details
- Lab reports and test results
- Imaging and radiology reports
- Hospital discharge summaries
- Patient-uploaded documents

<Warning>
  All medical data is encrypted and complies with healthcare data protection regulations.
</Warning>

### 4. AI-Aided Diagnosis

Get intelligent diagnosis suggestions based on:

- Current symptoms/complaints
- Complete patient medical history
- Prescription patterns
- Lab test results

```tsx
// Example AI diagnosis output
{
  diagnosis: "Viral Fever (URI)",
  confidence: 87,
  reasoning: "Symptoms align with recent prescription history (Paracetamol, Dec 15). 
             Seasonal pattern noted."
}
```

### 5. Patient Health Assistant

RAG-powered chatbot for instant patient information:

**Example Queries:**
- "Give me a summary of Sarah's health"
- "Any drug allergies?"
- "What medications is this patient currently taking?"
- "Show me recent lab results"

<Tip>
  The AI assistant retrieves information from the complete medical history and presents it in a conversational format.
</Tip>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Who can use Aria?">
    Aria is designed for both healthcare providers (doctors, clinics) and patients. Healthcare providers use it to create and manage digital prescriptions, while patients use it to access and manage their complete medical history.
  </Accordion>
  
  <Accordion title="Is Aria ABHA-compliant?">
    Yes! Aria is built to integrate seamlessly with India's Ayushman Bharat Digital Mission (ABDM) and supports ABHA-linked health records for full interoperability across the healthcare ecosystem.
  </Accordion>
  
  <Accordion title="How secure is my medical data?">
    Security is our top priority. All medical data is encrypted, stored securely, and complies with healthcare data protection regulations. Access is role-based and requires proper authentication.
  </Accordion>
  
  <Accordion title="Can I upload existing medical records?">
    Yes! Patients can upload lab scans, hospital reports, imaging results, and any other medical documents to build a complete digital health record.
  </Accordion>
  
  <Accordion title="What makes Aria different from other EMR systems?">
    Aria combines AI-powered features (voice dictation, smart medication search, AI-aided diagnosis) with ABHA integration and a patient-first approach. It's designed specifically for India's healthcare ecosystem with features like 100,000+ ABHA-compliant medication database.
  </Accordion>
